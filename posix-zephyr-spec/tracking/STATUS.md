## 依赖关系

```
FD表 ← 设备管理 ← 文件系统/网络
  ↓
PCB系统 ← 进程管理 ← Thread-as-Process
  ↓
线程管理 ← 信号机制
  ↓
内存管理 ← ELF加载器 ← 系统调用
```
