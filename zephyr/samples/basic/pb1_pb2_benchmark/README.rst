PB1 PB2 benchmark suite
#######################

该样例基于 ``PB1 -> PB2`` 连通性测例，增加以下度量：

- PB1/PB2 连通性与稳定性检查
- 直接在 ``TIM2`` 中断处理函数内采样的中断响应延迟
- 基于 ``k_timer`` 的调度唤醒延迟
- 启动时间度量（从 ``main()`` 入口到样例打印阶段）
- 镜像体积与 RAM 占用度量（通过构建输出中的 ``FLASH/RAM`` 汇总）

说明
****

- 中断响应延迟为“定时器更新事件 -> NVIC 进入 -> ISR 内首个时间戳”的近似值。
- 调度延迟为“期望唤醒时间 -> 定时器回调释放信号量 -> 线程恢复执行”的近似值。
- 启动时间是软件视角下从 ``main()`` 开始的时间，不包含上电到 ``Reset_Handler`` 之前的硬件过程。
